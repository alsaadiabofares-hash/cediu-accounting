📊 نظام إدارة سلف الموظفين

المميزات الرئيسية:

1. إدارة الحسابات:
  • ✅ إضافة سلفة جديدة للموظفين
  • ✅ تعديل بيانات السلفة (المبلغ، التاريخ، الحالة، الملاحظات)
  • ✅ حذف السلفة مع تأكيد العملية
  • ✅ عرض جميع السلف في جدول تفاعلي
2. الإحصائيات المالية:
  • 📈 إجمالي جميع السلف
  • 👥 عدد الموظفين المستفيدين
  • 📊 متوسط قيمة السلفة
  • 🏆 أعلى قيمة سلفة مسجلة
3. التصدير إلى Excel:
  • 📁 تصدير الإجماليات: ملف Excel يحتوي على ملخص مالي شامل
  • 📁 تصدير التفاصيل: ملف Excel متعدد الأوراق يحتوي على:
    • ورقة بجميع السلف
    • ورقة بحسابات الموظفين المجمعة
    • ورقة منفصلة لكل موظف مع إجمالياته الخاصة
4. المميزات التقنية:
  • 💾 حفظ البيانات تلقائياً في المتصفح (LocalStorage)
  • 🔍 بحث فوري في السلف
  • 🎨 تصميم عصري مع تأثيرات حركية
  • 📱 متجاوب مع جميع الأجهزة
  • ⌨️ اختصارات لوحة المفاتيح (Ctrl+N للإضافة، Escape للإغلاق)

كيفية الاستخدام:

  1. لفتح التطبيق: اضغط على الرابط أدناه لفتح الملف في المتصفح
  2. إضافة سلفة: اضغط على زر "إضافة سلفة جديدة" أو Ctrl+N
  3. التصدير: استخدم أزرار التصدير في الأعلى للحصول على ملفات Excel

الطريقة 1: التثبيت المباشر (PWA) - الأسهل

  1. انقل ملف HTML إلى هاتفك (عبر Bluetooth, WiFi Direct, أو Telegram)
  2. افتح الملف في متصفح Chrome أو Samsung Internet
  3. ستظهر رسالة تلقائية في الأسفل "تثبيت التطبيق" - اضغط عليها

  2. افتح الرابط في متصفح الهاتف

✅ نظام المحاسبة المالية المتكامل

📊 المميزات الجديدة:

1. نظام محاسبي متكامل (مدين | دائن | رصيد)

  • المدين (Debit): السلف المستحقة على الموظفين (باللون الأحمر)
  • الدائن (Credit): المدفوعات والسدادات (باللون الأخضر)
  • الرصيد (Balance): الرصيد التراكمي لكل موظف (يحسب تلقائياً)

2. استيراد من Excel 📥

  • سحب وإفلات ملف Excel مباشرة
  • اختيار يدوي للملف
  • اكتشاف تلقائي للأعمدة (اسم الموظف، المبلغ، النوع، التاريخ، البيان)
  • يدعم صيغ .xlsx و .xls
  • يدعم تواريخ Excel المختلفة

3. كشف حساب تفصيلي 📋

  • عرض جميع حركات الموظف مرتبة زمنياً
  • رصيد تراكمي يحسب بعد كل حركة
  • إمكانية تعديل أو حذف من داخل الكشف
  • تصدير الكشف إلى Excel منفصل

4. إدارة سريعة ⚡

  • إضافة سريعة: سلفة أو سداد بضغطة واحدة من الجدول
  • بحث فوري في جميع الحركات
  • تعديل مباشر لأي حركة

🚀 كيفية الاستخدام:

الاستيراد من Excel:

  1. جهز ملف Excel بهذه الأعمدة:
    • A: اسم الموظف
    • B: المبلغ
    • C: النوع (مدين/دائن) أو (سلفة/سداد)
    • D: التاريخ (اختياري)
    • E: البيان/الوصف (اختياري)
  2. اسحب الملف إلى منطقة الاستيراد أو اضغط "اختيار ملف"

إضافة حركة يدوياً:

  1. اضغط "حركة جديدة"
  2. اختر النوع: مدين (سلفة) أو دائن (سداد)
  3. أدخل البيانات واضغط حفظ

عرض كشف حساب:

  1. اضغط على أيقونة العين 👁️ بجانب اسم الموظف
  2. ستظهر جميع حركاته مع الرصيد التراكمي
  3. يمكنك تصدير الكشف إلى Excel

plain
Copy

2. اختر "New repository"
3. اسم المستودع: cediu-app
4. اجعله Public (عام)
5. اضغط "Create repository"

3. رفع ملف التطبيق

plain
Copy

2. اسحب ملف cediu_accounting_system.html إلى المربع
3. انتظر حتى ينتهي الرفع
4. اكتب في "Commit changes": "Initial upload"
5. اضغط "Commit changes"

4. تفعيل GitHub Pages

plain
Copy
1. اذهب إلى "Settings" (الإعدادات)
2. من القائمة اليسرى، اختر "Pages"
3. في "Source"، اختر "Deploy from a branch"
4. اختر الفرع "main" والمجلد "/ (root)"
5. اضغط "Save"
6. انتظر 2-5 دقائق
7. ستظهر رسالة: "Your site is published at https://username.github.io/cediu-app/"
